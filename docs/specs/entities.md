# エンティティ一覧

タスク一覧から抽出したエンティティとその関係。
ビュー設計の前提として整理する。

## エンティティ

| エンティティ | 概要 |
|---|---|
| レース場 | 競馬場の情報 |
| レース情報 | 特定のレースの情報 |
| レース結果 | レースの着順など結果 |
| 馬 | 出走馬の情報 |
| 騎手 | 騎手の情報 |
| 購入馬券 | ユーザーが購入した馬券 |
| 払い戻し | 購入馬券に対する払い戻し金額 |

## エンティティ間の関係

```
レース場
  └─ 1:N レース情報
              └─ 1:N レース結果
              |         ├─ N:1 馬
              |         └─ N:1 騎手
              └─ 1:N 購入馬券
                        └─ 1:1 払い戻し
```

## メモの位置づけ

メモはエンティティに付随する補足情報。独立したエンティティとして管理する。

| メモ | 紐づくエンティティ |
|---|---|
| レース場メモ | レース場 |
| 馬メモ | 馬 |
| 騎手メモ | 騎手 |
